##  wellcome to use vim <br>
vi 是 unix 家族下最功能強大的文字編輯器，讓用戶只要使用一個鍵盤就可以完成所有的編輯。而 vim 則是 vi 的加強版，甚至在 Windows 上也找得到 vim 的芳蹤。但 vi/vim 眾多的指令卻經常令初學者卻步，它的指令還有分大小寫，以下就是我所整理出來那些令人卻步的指令:

(每一列前面若有 * 表示為常用的指令)

[參考資料](http://www.vixual.net/blog/archives/234)



##  編輯模式

| |指令|說明
|--|--|--|
|*  |**i**|**在游標位置進入編輯模式**|
|   |I|在游標行的第一個非空白字元進入編輯模式|
|*  |a|在游標位置後進入編輯模式|
|*  |A|在游標行的最後一個字元進入編輯模式|
|*  |o|向下新增一行，並進入編輯模式|
|   |O|向上新增一行，並進入編輯模式|
|   |cc|刪除游標行，並進入編輯模式|
|*  |ESC|取消指令或退出編輯模式|

##  游標移動

| |指令|說明|

##  搜尋與取代

| |指令|說明|
|--|--|--|
|*|	/搜尋字串	|向下搜尋字串|
|*|	*	|將游標移到字串上，直接按 "*" 也可以做向下搜尋|
|*|	:set ic	|搜尋時不分大小寫|
|*|	:set noic|	搜尋時要分大小寫|

|*|	N	|繼續上一個搜尋結果|
|*|	:起始行,終止行s/搜尋字串/取代字串/gic	|從第 n 行到第 n 行取代字串 (後面的 g: 整行全部 i: 不分大小寫 c: 詢問)|
|*|	:1,$s/搜尋字串/取代字串/gic	|全部取代字串 (後面的 g: 整行全部 i: 不分大小寫 c: 詢問)|

##  vim 的設定檔
通常我會編輯 ~/.vimrc，在檔案最後加入:

**顯示列號**<br>
``` 
set number
```
**語法高亮度顯示**<br>
```
syntax on
```
**標記搜尋到的字串**<br>
```
set hlsearch
```
**自動縮排**<br>
```
set autoindent
```
**顯示說明**<br>
```
set ruler
```
**顯示編輯狀態**<br>
```
set showmode
```
**設定註解的顏色**<br>
```
highlight Comment ctermfg=cyan
```
**設定搜尋到的字串顏色**<br>
```
highlight Search term=reverse ctermbg=4 ctermfg=7
```
**設定 tab 鍵的字元數**<br>
```
set tabstop=4
```
